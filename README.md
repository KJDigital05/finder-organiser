## finder-organiser
A Python utility for automatically organising files on macOS.

## Features

- Sort files by extensions
- Custom organisational rules (planned)
- Duplicate detection using SHA-256 (planned)
- GUI Interface (planned)

## Why I Built This

My Finder folders, Desktop and Downloads directories had become difficult to manage and organise, so i decided to build a tool that could help automate this process whilst also improving my Python and software engineering skills.
